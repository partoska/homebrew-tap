# Contributing

Formulas in this tap are updated automatically by CI when a new release is published. Manual edits to formulas are generally not needed.

To contribute to the tools themselves, visit their source repositories:

- **p6a** — [partoska/p6a-cmd](https://github.com/partoska/p6a-cmd)

To report a packaging issue (broken formula, wrong sha256, failed install), open an issue in the relevant source repository.
