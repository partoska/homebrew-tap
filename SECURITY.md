# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Security issues in the packaged tools should be reported to the relevant source repository:

- **p6a** — [partoska/p6a-cmd](https://github.com/partoska/p6a-cmd/security/advisories/new)

Issues with the tap itself (e.g. incorrect sha256, tampered formula) can also be reported via the link above.
